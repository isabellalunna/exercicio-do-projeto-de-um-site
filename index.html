HTML5 e CSS3

HTML é uma das linguagens que usamos para desenvolver sites. A sigla HTML vem de Hypertext Markup Language.

HTML é a linguagem básica da Internet. Ele foi criado para tornar mais fácil para humanos e máquinas entenderem, como o Google ou outros sistemas que cruzam a Internet para capturar informações

A principal função do CSS3 é cancelar imagens de plano/Background , bordas arredondadas, introduzir transições e efeitos para criar vários tipos de animações, como relógios analógicos simples.

JavaScript

É uma linguagem de programação de comportamento que permite criar conteúdo dinâmico, controles de mídia e animações para tornar seu site mais interativo e interessante.

JS ou JavaScript é uma linguagem de programação de alto nível, criada em meados da década de 1990, ou mais precisamente pelo lendário programador Brendan Eich em 1996. Além de criar JavaScript, ele também é um dos fundadores da Mozilla Firefox.
